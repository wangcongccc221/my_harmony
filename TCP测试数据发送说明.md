# TCP测试数据发送说明

## 使用网络调试助手发送测试数据

你可以使用任何TCP调试助手（如网络调试助手、TCP/UDP调试工具等）连接到Harmony应用的TCP服务器，然后发送测试数据。

## 数据格式

TCP数据包格式为：
```
[SYNC字符串(4字节)] + [命令头(12字节)] + [数据体(StStatistics结构体二进制)]
```

### 1. SYNC字符串（4字节）
```
ASCII: "SYNC"
十六进制: 53 59 4E 43
```

### 2. 命令头（12字节，小端字节序）
```
源ID（4字节，int32，小端）
目标ID（4字节，int32，小端）
命令ID（4字节，int32，小端，统计信息命令=0x0001）
```

**示例（FSM1发送给上位机）：**
- 源ID: 0 (FSM1) = `00 00 00 00`
- 目标ID: 1 (上位机) = `01 00 00 00`
- 命令ID: 0x0001 (统计信息) = `01 00 00 00`

**完整的命令头（十六进制）：**
```
00 00 00 00 01 00 00 00 01 00 00 00
```

### 3. 数据体（StStatistics结构体二进制）

**结构体字段顺序和大小（小端字节序）：**

| 字段 | 类型 | 大小 | 说明 |
|------|------|------|------|
| nGradeCount[128] | ulong[] | 8字节×128=1024 | 各等级个数 |
| nWeightGradeCount[128] | ulong[] | 8字节×128=1024 | 各等级重量(克) |
| nExitCount[20] | ulong[] | 8字节×20=160 | 各出口个数 |
| nExitWeightCount[20] | ulong[] | 8字节×20=160 | 各出口重量(克) |
| nTotalCount | ulong | 8字节 | 总个数 |
| nWeightCount | ulong | 8字节 | 总重量(克) |
| nSubsysId | int | 4字节 | 子系统ID (0=FSM1, 1=FSM2) |
| nBoxGradeCount[128] | int[] | 4字节×128=512 | 各等级箱数 |
| nBoxGradeWeight[128] | int[] | 4字节×128=512 | 各等级每箱重(克) |
| nTotalCupNum | int | 4字节 | 总果杯数 |
| nInterval | int | 4字节 | 间隔数 |
| nIntervalSumperminute | int | 4字节 | 分选速度(个/分钟) |
| nCupState | ushort | 2字节 | 果杯状态 |
| nPulseInterval | ushort | 2字节 | 脉冲间隔(ms) |
| nUnpushFruitCount | ushort | 2字节 | 遗漏个数 |
| nNetState | quint8 | 1字节 | IPM网络状态 |
| nWeightSetting | quint8 | 1字节 | 重量整定标志 |
| nSCMState | quint8 | 1字节 | SCM状态 |
| nIQSNetState | quint8 | 1字节 | 糖度传感器状态 |
| nLockState | quint8 | 1字节 | 锁定状态 |
| ExitBoxNum[20] | int[] | 4字节×20=80 | 各出口箱数 |
| fSelectBasisRange[16] | StRange[] | 8字节×16=128 | 分选基准范围 |

**总数据体大小：** 1024 + 1024 + 160 + 160 + 8 + 8 + 4 + 512 + 512 + 4 + 4 + 4 + 2 + 2 + 2 + 1 + 1 + 1 + 1 + 1 + 80 + 128 = **3620字节**

## 完整数据包示例（十六进制）

### 最小测试数据包（所有字段为0）

**SYNC（4字节）：**
```

```

**命令头（12字节）：**
```

01 00 00 00  // 目标ID = 1 (上位机)
01 00 00 00  // 命令ID = 0x0001 (统计信息)
```

**数据体（3620字节，全0）：**
```
00 00 00 00 00 00 00 00 ... (重复3620次)
```

## 使用步骤

1. **启动Harmony应用的TCP服务器**
   - 确保应用已启动并监听端口（默认11279）

2. **打开网络调试助手**
   - 选择TCP客户端模式
   - 输入服务器IP（如：127.0.0.1或设备IP）
   - 输入端口（如：11279）
   - 点击连接

3. **发送数据**
   - 选择"十六进制"发送模式
   - 输入完整的十六进制数据包
   - 点击发送

## 简化测试方法

如果觉得构造完整数据包太复杂，可以先发送一个最小数据包（所有字段为0），验证连接和解析是否正常。

**最小测试数据包（十六进制）：**
```
53 59 4E 43 00 00 00 00 01 00 00 00 01 00 00 00
```
然后加上3620个`00`字节（数据体全0）。

## 注意事项

1. **字节序**：所有整数都是小端字节序（Little-Endian）
2. **数据对齐**：结构体是紧凑布局（无对齐）
3. **浮点数**：fSelectBasisRange中的浮点数使用IEEE 754单精度格式（小端）
4. **数组顺序**：数组元素按顺序连续存储

## 验证

发送数据后，检查：
1. Harmony应用是否接收到数据
2. UI界面是否更新（分选信息卡片、出口卡片等）
3. 日志中是否有解析成功的消息
